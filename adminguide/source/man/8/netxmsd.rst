netxmsd
=======

Synopsis
--------

**netxmsd** [*options*]

Description
-----------

Options
-------

-e              Run database check on startup
-c <file>       Use alternate configuration file. By default, netxmsd will try
                following files in order: */etc/netxmsd.conf*, *$PREFIX/etc/netxmsd.conf*
